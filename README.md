# GoRotas – Landing Page

Landing page institucional do **GoRotas**, plataforma que digitaliza e organiza o transporte alternativo de passageiros em vans e carros de lotação.

## Sobre o projeto

Página estática desenvolvida em HTML + CSS puro, sem dependências externas ou frameworks. Apresenta a proposta de valor do app para passageiros, motoristas e empresas de transporte.

## Estrutura

```
├── index.html        # Página principal
├── images/           # Imagens e mockups
│   ├── App Mockup.png
│   ├── LogoGoRotas (4) 1.png
│   ├── Map Background.png
│   └── Vans.png
└── README.md
```

## Seções da página

- **Hero** – Proposta de valor principal com CTA
- **Highlights** – Segurança, organização e praticidade
- **Conheça a GoRotas** – Funcionalidades para passageiros e motoristas
- **Como Funciona** – Passo a passo para passageiro e motorista
- **Diferenciais** – O que diferencia a plataforma
- **Benefícios** – Por perfil de usuário (passageiro, motorista, empresa)
- **Impacto Social** – Mobilidade urbana e formalização
- **Nossa História** – Origem e motivação dos fundadores
- **Mercado** – Dados do setor de transporte alternativo no Brasil
- **Cidades Disponíveis** – Maranhão e Piauí
- **Depoimentos** – Relatos antes/depois de usuários reais
- **FAQ** – Perguntas frequentes
- **Escassez** – Vagas de motoristas fundadores
- **CTA Final** – Chamada para download do app

## Como rodar

Abra o arquivo `index.html` diretamente no navegador. Não requer servidor ou instalação.

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações, scroll reveal via Intersection Observer)
- JavaScript vanilla (FAQ accordion, navbar scroll, animações de entrada)
- Google Fonts – Inter
